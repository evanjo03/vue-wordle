# vue-wordle

Wordle built to help learn Vue@2.

## Deployment

Found at: https://evanjo03.github.io/vue-wordle/

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
